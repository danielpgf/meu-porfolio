# meu-portfolio

## Portfólio Profissional

Projeto desenvolvido para a disciplina de Introdução à Computação, com o objetivo de criar um portfólio digital para apresentar informações pessoais, habilidades e projetos.

## Tecnologias
* HTML5
* CSS3
* JavaScript
* Git e GitHub

## Funcionalidades
* Apresentação pessoal
* Exibição de habilidades
* Galeria de projetos
* Contato profissional
